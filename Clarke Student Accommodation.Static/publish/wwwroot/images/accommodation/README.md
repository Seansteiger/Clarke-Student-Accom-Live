# Accommodation Images

## Instructions

Place your accommodation photos in this directory. The Gallery page will automatically display them.

### Supported Formats
- .jpg / .jpeg
- .png
- .gif
- .webp

### Recommended Image Types
1. **Bedroom photos** - Show furnished rooms, beds, study desks
2. **Kitchen photos** - Common kitchen areas, appliances
3. **Bathroom photos** - Clean, well-lit bathroom spaces
4. **Common areas** - Lounges, study spaces, social areas
5. **Exterior photos** - Building exterior, entrance, parking
6. **Security features** - Gates, burglar bars, controlled access
7. **Study spaces** - Quiet areas for studying
8. **General amenities** - Any other facilities

### Image Guidelines
- Use high-quality images (minimum 1200px width)
- Ensure good lighting
- Keep images clean and professional
- Show the accommodation in its best light
- Include diverse views of different areas

### Example Filenames
- bedroom-single.jpg
- bedroom-shared.jpg
- kitchen-main.jpg
- bathroom-1.jpg
- exterior-front.jpg
- common-area-lounge.jpg
- study-room.jpg

Once you add images here, they will automatically appear in the Gallery page!
